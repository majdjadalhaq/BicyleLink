
Title: Forgot Password & Reset Flow

Description:
Hi team,

I implemented the "Forgot Password" feature, completing the core authentication system.

What's New:
- **Forgot Password Page**: Users can request a reset code via email.
- **Reset Password Page**: Users can verify the code and set a new password.
- **Login Link**: Added "Forgot Password?" link to the login form.
- **Security**:
  - Anti-enumeration on reset request.
  - Code expiry (15 mins).
  - One-time use code (prevent replay).
  - Secure password hashing.

Backend:
- `POST /users/request-reset`
- `POST /users/reset-password`

Frontend:
- `ForgotPassword.jsx`
- `ResetPassword.jsx`
- `App.jsx` routes.

Testing:
- `server/test-password-reset.js`: Full integration test passed (Request -> Reset -> Login).

This PR completes the Authentication System.
